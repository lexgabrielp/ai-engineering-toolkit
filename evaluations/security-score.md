# Security Score

Score each category from 1 to 10.

## Categories

- Correctness
- Maintainability
- Security
- Performance
- Test Coverage
- Observability
- Scalability
- Developer Experience

## Result

- PASS: no critical issues, average score >= 8
- WARN: no critical issues, average score 6–7
- FAIL: critical issue exists or average score < 6

## Output Format

```txt
Overall: PASS/WARN/FAIL
Score: X/10
Critical Issues:
Recommended Fixes:
```

# Contributing

Thanks for your interest in contributing to AI Engineering Toolkit.

## What You Can Contribute

You can contribute:

- New agents
- New skills
- New workflows
- New evaluation rubrics
- Tool adapters
- Documentation improvements
- Example usage patterns

## Contribution Guidelines

Please keep contributions:

- Reusable across projects
- Clear and easy to understand
- Tool-agnostic when possible
- Production-minded
- Focused on helping developers work better with AI coding assistants

# AI Feature Chain

Use this sequence for LLM/RAG/agentic features:

1. `ai-engineer` designs the AI flow.
2. `rag-engineer` designs retrieval/context.
3. `prompt-engineer` defines prompt contracts.
4. `ai-evaluation` skill defines evals.
5. `security-engineer` checks prompt injection and data leakage.
6. `code-reviewer` validates implementation.

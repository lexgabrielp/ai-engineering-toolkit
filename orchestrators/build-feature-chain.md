# Build Feature Chain

Use this sequence for serious feature work:

1. `software-architect` designs the approach.
2. `senior-backend-engineer` or `react-engineer` implements.
3. `test-engineer` adds coverage.
4. `security-engineer` reviews risks.
5. `performance-engineer` checks bottlenecks.
6. `code-reviewer` performs final review.

Final output must include:
- Implementation summary
- Tradeoffs
- Test plan
- Risks

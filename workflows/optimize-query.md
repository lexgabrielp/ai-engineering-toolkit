# Optimize Query Workflow

## Steps

1. Understand the goal and constraints.
2. Inspect existing project conventions.
3. Propose a minimal implementation plan.
4. Implement in small, reviewable changes.
5. Add or update tests.
6. Review security, performance, and maintainability.
7. Summarize changes and follow-up tasks.

## Required Output

- What changed
- Why it changed
- How to test it
- Risks or tradeoffs
